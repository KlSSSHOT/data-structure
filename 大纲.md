###1.3实践 最大子列和
给定N个整数的序列{A1,A2,A3,....An}求函数的最大连续子列和
1.算法一 求出所有子列,算出最大子列
2.算法二 分而治之先一分为二,求左右以及连续的最大值,得到最大的一个return出来
3.算法四 在线处理 
```cpp
int sum,maxsum;
int i;
sum=maxsum=0;
for(i=0;i<N,i++){
    sum+=a[i];
    if(sum>maxsum) maxsum=sum;
    else if(sum<0) sum=0;
}
```
- 为什么叫在线处理:"在线"的意思是指每输入一个数据就能进行及时处理,在任何一个地方中止输入,算法都能正确的给出当前的解
